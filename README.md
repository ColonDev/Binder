Binder
======

Binder is a Spring Boot revision platform for managing and sharing content with attachments.

Tech stack
----------
- Java + Spring Boot
- JPA/Hibernate
- JUnit 5 for tests

Getting started
--------------
1) Ensure Java is installed (17+ recommended).
2) Ensure Maven is available on your PATH.
3) Configure your database settings in `application.properties` if needed.

Run the app
-----------
```bash
mvn spring-boot:run
```

Run tests
---------
```bash
mvn test
```

Notes
-----
- The repository includes `mvnw`, but the Maven wrapper files are not checked in.
- If you prefer to use the wrapper, add the `.mvn/wrapper` directory.
